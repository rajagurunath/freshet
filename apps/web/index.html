Open source · local-first · one agent per company

Everything your team taught the AI, in one place.

Context Hub captures the AI coding sessions on every laptop, lets each person curate and redact what matters, and turns the rest into one company agent anyone can ask.

Get the desktop app macOS · Linux · Windows

Parsing happens on-device. Nothing leaves a laptop until someone chooses to push — and secrets are redacted first.

Your team's best thinking is sitting in .jsonl files.

Claude Code, Codex, and Kilo Code transcripts are parsed entirely on your machine. Browse every session, see what was actually decided — including what /compact threw away — and push only what you choose, as a full transcript or a curated summary.

~/.claude · ~/.codex · kilocode tasks

One agent that has actually read everything.

Hybrid retrieval — full-text and vector search, rank-fused — finds the exact error string and the half-remembered decision. Every answer cites the sessions it came from, so trust is checkable, not assumed.

POST /v1/query · mode: hybrid · citations included

Every pull request, with its reasoning attached.

One command mints a token-gated context page from the session behind a PR: why it exists, the decision trail, the dead ends that were tried first. Reviewers read the thinking, not just the diff.

contexthub-pr 142 → agent-context link on the PR

Sessions connect into a map of your company.

Repos, services, features, people, and decisions are extracted into a knowledge graph. When two teams touch the same feature from different repos, the graph says so — and the agent uses it to answer cross-cutting questions.

nodes: repo · service · feature · person · decision

Skills and prompts, shared like packages.

Push the skills, commands, and agents in your local ~/.claude to the hub; pull what your teammates built. The hub speaks the OpenSharing protocol, so any compatible client can read the catalog too.

GET /opensharing/shares · Agent Skill + Volume

Set it once. The hub stays current.

A file watcher notices finished sessions and pushes them after a quiet period — redacted, deduplicated, idempotent. Summarization runs nightly on the Batch API at half price, or on a local model for free.

watch → quiet 5 min → push · summarize at 02:00

Run it this afternoon.

Two pieces: a FastAPI hub your team points at, and a desktop app each person runs. No SaaS, no data leaving your infrastructure.

git clone <your-fork>/context-hub && cd context-hub
make api-install && make api    # the hub, on :8787
make desktop-install && make tauri    # the desktop app
Full setup in README.md at the repo root · OpenSharing-compatible surface · API-key auth · S3 or local disk

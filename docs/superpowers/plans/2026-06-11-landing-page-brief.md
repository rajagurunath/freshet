# Context Hub Landing Page — Design Brief (from competitor research, 2026-06-11)

Sources studied live: conductor.build (primary reference), linear.app, cursor.com, resend.com, braintrust.dev, warp.dev, Augment Code, graphite.dev.

Key takeaways folded into this brief: Conductor — let the real product UI carry all visual weight (interactive window mock, monochrome page); Linear — numbered sub-variant switchers inside feature sections, scroll-reveal screenshots; Cursor — warm off-white page, full-bleed pull quotes; Resend — code/terminal snippet as instant proof; Warp — install command inline in the hero; Augment — "named signals" problem framing before any feature pitch; Graphite — case-study-style social proof.

## Constraints (DESIGN.md compatible)
Colors: `--bg #FBFAF8`, `--ink #1A1815`, `--accent #F2541B`, `--accent-wash #FCEDE6`, `--border #E7E3DB`, `--bg-sunken #F4F2EE`, `--bg-elevated #FFF`. No gradients, no glassmorphism, no drop shadows; hairline borders. Inter (tight `-0.02em` heading tracking), JetBrains Mono for code/session content. Body 15px; section headings 28–40px; display headline 52–60px. **Page stays warm-paper light; only the product UI mock uses a dark app shell (`#1C1B18`/`#252320`) — that light/dark contrast is the look.**

## Animation table
Hero headline/subhead/CTAs: load fade + translateY(8px→0), 400ms ease-out, 100/250/400ms delays. Hero window: fade + scale .97→1, 500ms, 300ms delay. Marquee: 40s linear infinite. Signal cards: scroll-reveal stagger 80ms. Steps: stagger 120ms. Feature cards: stagger 60ms. Flow arrows: dashoffset draw 400ms, stagger 200ms; node pulse 300ms after. Hovers: 120–150ms ease-out. Nav bg: 200ms after 80px. All scroll triggers via IntersectionObserver; respect `prefers-reduced-motion` (disable everything, show final state). **NO** scroll-jacking, parallax, video, springs, looping demos.

## Copy rules
Subject-verb-object headlines. Name the tools (Claude Code, Codex, Kilo Code) — specificity = credibility. Privacy stated in product copy, not footnotes. Orange accent: CTAs, active states, ≤1 key word per headline. Body blocks ≤40 words. Numbers over adjectives.

## Anti-patterns
(1) AI-washed generic copy ("Unlock insights", "Transform your workflow") — every sentence must only describe Context Hub. (2) Feature-first ordering — pain before features. (3) Dark-mode page — keep warm paper white; dark belongs only inside the product mock.
